USAGE INSTRUCTIONS

Quick Start (Assignment 1):
  1. Install dependencies:
     pip install -r requirements.txt
  
  2. Configure dataset (data.yaml):
     - Set paths to train/val/test images
     - Define classes
  
  3. Train model:
     from thermal_detector import ThermalObjectDetector
     detector = ThermalObjectDetector('config.yaml')
     detector.load_model('yolov8n.pt')
     detector.train('data.yaml', epochs=100)
  
  4. Run inference:
     detections = detector.detect('thermal_image.jpg', visualize=True)
  
  5. Evaluate:
     results = detector.evaluate('test/images', 'test/labels')
